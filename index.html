4 Building Blocks
Sections, Containers, Rows, and Wrappers

Because things need to be simple but not simplistic.

Sections
section

These are for separating sections of the page. They provide full width backgrounds and horizontal spacing.

Containers
.xy-container

Containers contain, that is what they do. They provide a max width and a gutter.

Rows
.xy-row

Rows are flexible. They give structure to columns of content. They are not containers but can be combined with them. They use CSS Grid because layout and structure.

Wrappers
.xy-wrapper

Wrappers are content focused. They are for alignment of elements within a column or row. They are flex!

Containers: .xy-container

Four different container widths. Containers contain stuff within them, they form the outer part of a layout.

Full width .xy-mw-max
Wide width .xy-mw-wide
Inner width .xy-mw-inner
Text width .xy-mw-text

Grid System: .xy-row and .xy-col

Use the power of CSS grid to make a consistent responsive layout quickly. Rows sit within containers (or are containers). Columns sit within rows
